# 🇿🇲 Zambian Fiscal Policy Visualizer

A  document parser built to analyse the **Mines and Minerals Development (Amendment) Act, 2016**. This tool extracts fiscal terms and royalty structures to increase data accesability in natural resource governance.

## ✊ The Purpose 
Legal jargon often acts as a barrier to inclusion. This app automates the extraction of:
- **Copper Royalty Tiers**: Visualizing the $4,500 and $6,000 price thresholds.
- **High-Value Minerals**: Flagging 6% flat rates for gemstones.
- **Accountability Clauses**: Identifying "Arm's Length" transaction rules.

## 🛠️ Tech Stack
- **Language:** Python
- **Interface:** Streamlit (Community Cloud)
- **Engine:** PyPDF2 + Regular Expressions (Regex)

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/ChikondiBanda1/Zambian-Fiscal-Policy-Visualizer`
2. Install dependencies: `pip install -r requirements.txt`
3. Launch the app: `streamlit run app.py`
